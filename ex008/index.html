Principais formatações

Negrito / Destaque

Há duas maneiras: semântica (significado) e não semântica (forma).

Nesta frase, temos um termo em negrito usando usando a tag <b> (não semântica).

Nesta frase, temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande (usando a tag <big>, atualmente obsoleta) e um texto pequeno (usando a tag <small>) neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído (usando a tag <del>) para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins> (semântica).

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O, usando a tag <sub>

Código-fonte

Exemplo de código em Python (mono-espaçado; tag <code>)

num = int (input('Digite um número')) if num % 2 == 0: print(f'O número {num} é par') else: print(f'O número {num} é ímpar') print ('FIM DO PROGRAMA')

Exemplo de código em Python (pré-formatado; tags <pre> e <code>)

        
num = int (input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é par')
else:
    print(f'O número {num} é ímpar')
print ('FIM DO PROGRAMA')
